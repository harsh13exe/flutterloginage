## 🚀 Getting Started

### Prerequisites

- Flutter SDK installed
- Android Studio / VS Code

### Installation

1. Clone this repository:
git clone https://github.com/harsh13exe/flutterloginage.git

2. Navigate to the project directory:
cd flutterloginage

3. Install dependencies:
flutter pub get

4. Run the app:
flutter run
